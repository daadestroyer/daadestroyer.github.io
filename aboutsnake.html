Snake Game using Arrays

posts

About Project

  • Project Type : Web Application
  • Technologies Used : HTML,CSS,JS
  • Team Size : Only Me
Play Game

Features :
  • Snake can move in left,right,up,down direction
  • Snake can move whenever snake eat apple points will be increased
  • When ever snake X and Y axis collide with HTML width or height it the game will be over
  • Snake is moving in all direction by shifting the last cell in respect with the grid axis
  • Suppose snake is moving in right direction so last cell of element will be pop and unshifted to the head of the array
  • After refreshing the game you will be able to play game again
