Admin Features
  • Admin can able to see total number of regsitered customers
  • Admin can able to add some bank branches
  • Admin can able to view all bank branches
  • Admin can able to get the account opening request of customers
  • Admin can take some action on customers request , admin can accept , delete , remove customers request
  • If admin accept the account opening request or delete the request user immediately get the notification
  • If admin accept the account opening request then he can able to see the account of that customer
  • Admin can check the deposit history of particular customer
  • Admin can able to check the withdrawal history of particular customer
  • Admin can able to check the transaction history of particular customer
    •

    Customer Features
  • Customer can regsitered her self in website
  • After registration customers can do login in the webiste
  • After login customers can able to send account opening request to admin
  • If the account open request of particular user deleted , accepted , removed then immediately customer get the notification
  • If the account open request of particular user accepted then user can able to see her account with balance 0.0
  • User can able to deposit some money in his account
  • User can able to see the deposit history of his account
  • User can able to withdrawal some money in his account
  • User can able to see the withdrawal history of his account
  • User can able to tranfer some money to other user account
  • User can able to see the transfer history of her account
  • Both the user who is tranfering and the user who is recieved money get the notification
  • The user who is tranfering money will get the debit notification
  • THe user who is recieved money will get the credit notification
